# Development Roadmap – AI Dreams World

Lộ trình phát triển dự kiến, có thể thay đổi khi có feedback.

---

## Phase 1 – Core Prototype (MVP)

- Dream Engine basic loop (Genesis + Evolution).
- 5 core runtime agents: 3 creative agents (Creator, Civilization, Storyteller) + Historian + Guardian.
- World state model + SQLite schema.
- Event feed API + Dream Feed UI.
- Simple Timeline + World Overview.

## Phase 2 – Expanded Simulation

- Thêm agents: Culture, Myth, Ecology, Economy, Chaos.
- Civilization simulation sâu hơn (rise/fall, alliances, basic trade).
- Culture & myth generation rõ ràng.
- Era summaries.

## Phase 3 – Visualization

- Region/world map.
- Civilization Viewer.
- Mythology Browser.
- Knowledge Graph.
- Export tools (wiki, annals, myths) hoàn thiện.

## Phase 4 – Advanced Simulation

- Ecosystem dynamics.
- Trade systems rõ hơn.
- AI civilizations với behavior phức tạp hơn.

## Phase 5 – Community & Multi-world

- Multi-world support.
- World forking & alternate timelines.
- Shared worlds gallery.
- Collaborative storytelling.
