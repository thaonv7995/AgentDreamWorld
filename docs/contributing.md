# Contributing – AI Dreams World

Cảm ơn bạn đã quan tâm đến AI Dreams World! Đây là vài hướng dẫn nhanh để đóng góp.

---

## 1. Những mảng cần giúp

- **Backend**
  - Dream Engine & simulation rules.
  - Agent Runtime & LLM integration.
  - World model & DB schema.

- **Frontend**
  - Dream Feed, Timeline, World Overview.
  - Map, graph, mythology UI.

- **Research / Design**
  - Simulation boundaries & world rules.
  - Agent behaviors & prompts.
  - UX cho việc khám phá thế giới.

---

## 2. Bắt đầu như thế nào

1. Đọc `README.md`, `ai-dreams-vision.md`, `MVP-SPEC.md` để hiểu project.
2. Xem `DEV-SETUP.md` để chạy project local.
3. Chọn một issue hoặc đề xuất tính năng mới.

---

## 3. Quy ước chung

- Code backend: Rust (fmt/clippy clean).
- Frontend: React (TS ưu tiên), code style consistent.
- Viết thêm/ cập nhật docs nếu chỉnh behaviour đáng kể.
- Tôn trọng triết lý project: **agent imagination first, world simulation coherent, user là viewer**.
