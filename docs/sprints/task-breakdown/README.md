# AI Dreams World – Sprint Task Breakdown Library

Thư mục này chứa **task breakdown docs** cho từng sprint.

Vai trò của lớp doc này:

- đứng giữa planning docs và implementation docs
- bẻ sprint thành work packages và task issue-ready
- giúp giao việc theo thứ tự triển khai, cut line và artifact nghiệm thu

Nguyên tắc:

- `docs/sprints/SPRINT-XX.md`: planning / BA sprint doc
- `docs/sprints/implementation/SPRINT-XX.md`: implementation companion
- `docs/sprints/task-breakdown/SPRINT-XX.md`: issue-ready breakdown

Mỗi file breakdown nên trả lời:

- Sprint này chia thành mấy work packages?
- Task nào là blocker/critical path?
- Task nào có thể defer nếu sprint bị trượt?
- Artifact nào dùng để nghiệm thu nhanh?
