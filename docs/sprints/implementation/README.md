# AI Dreams World – Sprint Implementation Library

Thư mục này chứa **implementation companion docs** cho từng sprint.

Mục đích:

- chuyển sprint planning docs sang góc nhìn thực thi kỹ thuật
- chốt implementation slices cho backend / frontend / runtime / ops
- làm handoff doc giữa planning, execution và review

Nguyên tắc:

- File trong `docs/sprints/` vẫn là planning/BA sprint docs.
- File trong `docs/sprints/implementation/` là implementation docs đi kèm.
- Mỗi implementation doc chỉ chốt:
  - architecture slice cần build
  - data/API/runtime contracts cần khóa
  - verification path tối thiểu
  - handoff sang sprint kế tiếp
